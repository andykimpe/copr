Info Utilisateur

Conformément au RGPD (Règlement Général sur la Protection des données 2016/679 de l’Union européenne), les utilisateurs ont le droit de voir toutes les données, que le service stocke à leur sujet, et aussi le droit de les supprimer. En savoir plus sur votre Droits et vie privée dans le projet Fedora.

Obtenir les Données

Il est possible de télécharger un fichier JSON avec un dump de toutes les informations que Copr connaît sur vous (par ex. liste de vos projets, vos builds, vos coordonnées, vos tokens API, etc).

Si vous avez des difficultés à générer un fichier JSON, SVP Contactez nous, et nous l'envérons sur l'émail de compte Fedora.

Supprimer mes Données

Cela supprimera les informations que Copr connaît sur vous, telles que le fuseau horaire et les jetons API. Cependant, votre nom d’utilisateur et Les e-mails ne peuvent pas être supprimés car ils sont liés à votre compte Fedora. De plus, si vous souhaitez supprimer vos builds/projets, vous devez le faire manuellement (éventuellement en utilisant les données acquises ci-dessus).

Vous pouvez toujours vous connecter ultérieurement. Cependant, vous devrez créer un nouveau jeton API.
